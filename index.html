The Effective Fragment Potential method implementation

Introduction

LIBEFP is a full implementation of the Effective Fragment Potential (EFP) method (see Gordon et al., Chem. Rev., 2012). LIBEFP is designed to allow developers an easy way to add EFP support to their favourite quantum chemistry software package.

Efpmd package

EFPMD is a molecular simulation package based on LIBEFP. It allows running EFP-only molecular simulations such as geometry optimization and molecular dynamics. EFPMD is a part of LIBEFP distribution. See README file for more information on how to use the package.

Libefp users

LIBEFP is currently being used in PSI (development version) and EFPMD packages.
